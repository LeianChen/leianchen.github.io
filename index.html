GEOG5990M: Programming for Geographical Information Analysts

About the Course:

GEOG5990M is one of the compulsory programming course for the MSc GIS programme. The course is designed to deliver the basics of programming using the Python programming language. It uses examples taken from spatial analysis and modelling to conduct scientific programming that analyses real-life problems. There are two assessments for this module: an online portfolio and an independent project.
